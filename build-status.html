🍎 Mac Steps (Copy the LifeStages-Mobile folder to your Mac)

1
Copy the LifeStages-Mobile folder to your Mac via USB drive, AirDrop, or Git
The entire project needs to be on the Mac for Xcode builds
2
Open Terminal, cd into the folder, run npm install
This installs all dependencies including Capacitor plugins
3
Run BUILD_TARGET=mobile npm run build
This creates the static bundle in the 'out' folder
4
Run npx cap add ios then npx cap sync
Creates the native iOS project and copies your build into it
5
Run npx cap open ios to open in Xcode
Select your Apple Developer Team, set bundle ID, then Cmd+R to run
6
When ready: Product → Archive → Distribute App
Upload to App Store Connect for review
💡 Important: You need an Apple Developer account ($99/year) to submit to the App Store. Sign up at developer.apple.com/programs if you haven't already. You also need Xcode 15+ installed from the Mac App Store.
